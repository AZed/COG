Earth System CoG is a web environment that enables users to create project workspaces, connect projects into networks, share and consolidate information within those networks, and seamlessly link to tools for data archival, reformatting and search, data visualization, and metadata collection and display. CoG is integrated with the Earth System Grid Federation (ESGF) data distribution software and provides an easy to use interface to its services.

Collaborative Project Workspaces

CoG workspaces come with their own wiki, news, data space, and customizable navigation. Projects can be linked to other projects, and information can be shared across this network.

Support for Project Governance

CoG supports open governance processes by encouraging projects to share information about their management bodies, organizational roles, prioritization strategies, meeting and call schedules, and other aspects of project operation.

Data Services

CoG comes bundled with an Earth System Grid Federation (ESGF) datanode. Individual projects can store data on the node and customize a search to their project's holdings.

Metadata Services

CoG has partnered with the ES-DOC project to provide services to create and display Earth system model metadata.
