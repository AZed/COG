Connecting Resources in the Earth Sciences

CoG is a multi-institutional project that seeks to examine, within the Earth sciences, the organizational characteristics of community software projects and to create software insfrastructure to support their effecient governance and operation. Learn More

Below is a list of CoG's features. Click on the yellow box to start exploring these capabilities.

Collaborative Project Workspaces

The heart of CoG are CoG project workspaces. Workspaces come with their own wiki, news, data space, and customizable navigation. Projects can be linked to other projects in a heirarchy of parents, peers, and children. Information can be shared across this network.

Project Governance

CoG allows projects to define their project governance through the description of management bodies, organizational roles, prioritization strategies, mission, vision, values, partners, etc.

Data Services

CoG comes bundled with an Earth System Grid Federation (ESGF) datanode. Individual projects can store data on the node and customize a search to their project's holdings.

Metadata Services

CoG has partnered with the ES-DOC project to provide services to create and display Earth system model metadata.
